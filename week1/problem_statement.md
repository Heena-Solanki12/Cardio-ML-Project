--> Problem Definition <--

Cardiovascular disease (CVD) is one of the leading causes of death worldwide. Early prediction using patient health indicators can help in preventive treatment.

This project aims to develop a machine learning model to predict whether a person is likely to have cardiovascular disease based on various health parameters.

--> Dataset Information <--

- Dataset: Cardiovascular Disease Dataset from Kaggle
- Contains: 70,000 records, 12 features
- Features include:
    - Age
    - Gender
    - Height
    - Weight
    - Blood Pressure systolic/diastolic
    - Cholesterol
    - Glucose
    - Lifestyle factors (smoking, alcohol, activity)
