# Cardio-ML-Project ❤️

Complete ML project predicting cardiovascular disease risk using custom Logistic Regression.

## Setup
1. Download dataset: https://www.kaggle.com/datasets/sulianova/cardiovascular-disease-dataset
2. `pip install -r requirements.txt`
3. `python model/train_model.py` (trains & saves model)
4. `streamlit run app.py`

## Features
- Custom Logistic Regression (no sklearn training)
- Multi-page Streamlit UI (light/dark theme)
- EDA, Predictions, Metrics dashboards
- Deployable on Raspberry Pi/Streamlit Cloud

**Demo Metrics**: Accuracy ~73%, ROC-AUC ~0.72[file:2]
